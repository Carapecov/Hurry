Sistema de Banco Hurry | Hurry Bank

**Este Projeto foi desenvolvido por mim com foco no aprendizado e aprofundamento da Linguagem C, Arquitetura de Software e Desenvolvimento lógico. Utilizando os seguintes conceitos**:

1.*Modularização: Separação de responsabilidades em cada arquivo, cada arquivo recebe uma tarefa e executa. Na arquitetura de software, é importante separar arquivos que façam cada um sua responsabilidade, porque além de ser mais prático de estruturar, também é mais técnico para resolver problemas do código ou fazer alterações*.

2.*Segurança: Implementação de validações para evitar saldo negativo e consistência dos dados*.

3.*Memória: Simulação de banco de dados usando Struct*.

**Tecnologias e conceitos**:

 1.*Linguagem C: Loop, vetor, gerenciamento de memória, struct e strings*.
 
 2.*Arquitetura de software: Separação por tarefas ou camadas(Interface, lógica e dados)*.

 **COMO FAZER FUNCIONAR?**

 **Abra o terminal do Visual Studio Code, clique em "New Terminal" depois:**
 1. *Compile os módulos*:
   `gcc main.c login.c transferencia.c validar.c -o hurry_bank`
 2. *Execute*: 
   `./hurry_bank`
   *E veja como funciona uma simulação de sistema bancário em C!*

 **Caso não se tenha o gcc, seria necessário baixá-lo ou jogar num compilador online como por exemplo, o online GDB para visualização e interação com o programa**
